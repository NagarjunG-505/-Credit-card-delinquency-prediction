Credit Card Delinquency Prediction Analysis

Project Overview

This project analyzes credit card customer data to identify factors that contribute to delinquency and payment defaults. The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and predictive modeling to understand customer behavior and credit risk.

Objectives

- Analyze customer payment patterns.
- Identify key factors leading to credit card delinquency.
- Build a predictive model for delinquency risk.
- Generate insights through data visualization.

Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

Dataset

The dataset contains customer demographic information, credit limits, bill amounts, payment history, and delinquency status.

Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation
7. Insights and Recommendations

Results

The model helps identify customers with a higher probability of delinquency, enabling better credit risk management and decision-making.
